# タスクスケジューラ タイムライン可視化ツール

Windowsタスクスケジューラの設定内容を取得し、**24時間タイムライン形式**でローカルHTMLに可視化するツールです。

## 動作環境

| 項目 | 内容 |
|------|------|
| OS | Windows 10 / 11 |
| Python | 3.7 以上（標準モジュールのみ使用） |
| 外部ライブラリ | 不要 |
| 出力 | ローカルHTML（`output.html`） |

## ディレクトリ構成

```
タスクスケジューラ管理/
 ├ main.py       # エントリポイント
 ├ parser.py     # schtasks取得・CSVパース・Trigger展開
 ├ model.py      # Task / Trigger / TaskInstance クラス定義
 ├ renderer.py   # HTMLタイムライン生成
 ├ 仕様書.md
 ├ README.md
 └ output.html   # 実行後に生成される
```

## 使い方

### 1. 管理者権限でコマンドプロンプトを開く

タスクスケジューラの情報取得に管理者権限が必要な場合があります。

### 2. スクリプトを実行

```cmd
cd C:\xampp\htdocs\ndma-tools\タスクスケジューラ管理
python main.py
```

### 3. output.html をブラウザで開く

実行後、同フォルダに `output.html` が生成されます。  
ダブルクリックするか、ブラウザにドラッグ＆ドロップして開いてください。

## 出力イメージ

```
00:00        06:00        12:00        18:00        24:00
  |            |            |            |            |
TaskA  [====]
TaskB              [=] [=] [=] [=] [=]
TaskC                        [============]
```

- **横軸**：0:00〜24:00（1440px固定）
- **縦軸**：タスク一覧
- **バー幅**：推定実行時間
- **バー位置**：開始時刻

## 機能一覧

| 機能 | 説明 |
|------|------|
| タイムライン表示 | タスクの実行タイミングを横バーで可視化 |
| 色分け | タスクごとに異なる色を自動割り当て |
| ツールチップ | バーにホバーすると開始時刻・実行時間を表示 |
| 重複積み上げ | 同じ時間帯の複数バーを縦方向にずらして表示 |
| フィルタ | タスク名の部分一致でタスク行を絞り込み |
| 凡例 | 画面下部にタスク名と対応色を一覧表示 |

## 処理フロー

```
[schtasks /query /v /fo csv]
        ↓ subprocess
    CSVパース (csv モジュール)
        ↓
    Task / Trigger 構造に変換
        ↓
    繰り返しを1日分の TaskInstance に展開
        ↓
    実行時間を推定（LastRun vs NextRun 差分 / fallback 60秒）
        ↓
    HTML生成 → output.html
```

## 実行時間の推定方法

1. **方法A（優先）**: `Last Run Time` と `Next Run Time` の差分から推定（5秒〜1時間の場合のみ採用）
2. **方法B（fallback）**: 推定不可の場合は固定値 **60秒** を使用

## 注意事項

- 管理者権限なしで実行した場合、一部のシステムタスクが取得できないことがあります。
- `Schedule Type` が取得できないトリガーはスキップされます。
- `WEEKLY` / `MONTHLY` トリガーは曜日・日付に関わらず、設定された時刻にその日実行されるものとして表示されます。
